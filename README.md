# Contador Angular

Projeto simples desenvolvido em Angular com o objetivo de praticar:

- Estrutura de componentes
- Binding de eventos `(click)`
- Interpolação `{{ }}` para exibição de dados
- Atualização de estado sem manipulação direta do DOM

## Funcionalidades

- Incrementar valor
- Decrementar valor
- Atualização automática da interface

## Tecnologias utilizadas

- Angular
- TypeScript
- HTML
- CSS

## Objetivo do projeto

Este projeto faz parte do meu aprendizado inicial em Angular, com foco em entender
como funciona a comunicação entre o template e a lógica do componente.

## Como executar o projeto

```bash
npm install
npm start
